# student-management-system-       # Student Management System

A console-based Student Management System built in C++.

## Description
This program manages student records using file handling.
Data is saved permanently in a text file.

## Features
- Add new student
- Display all students
- Delete student by ID
- Menu-driven interface
- Data saved in `students.txt`

## Technologies Used
- C++
- File Handling (fstream)

## How to Run
```bash
g++ student_mgmt_short.cpp -o sms
./sms
```
